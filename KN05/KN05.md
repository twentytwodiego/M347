# KN05

## A) Bind mounts

Befehle:
```bash
mkdir C:\KN05
cd C:\KN05
notepad script.sh
docker run -it --name KN05 -v C:\KN05:/app ubuntu bash
```

`-v` steht für Volume / Mount, verbindet Speicher vom Host mit Container

`C:\KN05` = Host Ordner

`/app` = Ordner im Container

[Screencast](Screencast.mp4)


## B) Volumes

```bash
docker volume create KN05B
docker run -it --name c1 -v KN05B:/data ubuntu bash
docker run -it --name c2 -v KN05B:/data ubuntu bash
```
